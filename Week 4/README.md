# **README – SOC Practical Implementation Report**

## **Project Title**

Security Operations Center (SOC) Practical Implementation and Analysis

---

## **Overview**

This project demonstrates the implementation of various Security Operations Center (SOC) activities, including threat hunting, SOAR automation, incident analysis, alert triage, and security monitoring. The objective is to simulate real-world cybersecurity operations using industry-standard tools and methodologies.

---

## **Tools and Technologies Used**

* **Elastic Security** – Log analysis and threat detection
* **Velociraptor** – Endpoint investigation and query execution
* **AlienVault OTX** – Threat intelligence lookup
* **Wazuh** – Security monitoring and alert detection
* **TheHive** – Incident response and case management
* **CrowdSec** – Automated threat blocking
* **VirusTotal** – Threat validation and analysis
* **Google Sheets / Draw.io** – Metrics calculation and diagram creation

---

## **Practical Tasks Covered**

### **1. Threat Hunting**

* Developed a hypothesis for detecting suspicious activity
* Analyzed logs using Elastic Security
* Used AlienVault OTX for threat intelligence
* Performed endpoint analysis using Velociraptor

---

### **2. SOAR Playbook Development**

* Designed an automated playbook for phishing detection
* Integrated tools for automated response
* Blocked malicious IPs using CrowdSec
* Created incident cases in TheHive

---

### **3. Post-Incident Analysis**

* Conducted Root Cause Analysis using the 5 Whys method
* Created a Fishbone Diagram
* Calculated SOC metrics (MTTD, MTTR, False Positive Rate)

---

### **4. Alert Triage and Automation**

* Analyzed alerts using Wazuh
* Validated threats using VirusTotal
* Automated alert handling using TheHive
