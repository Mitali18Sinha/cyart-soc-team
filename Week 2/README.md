# CYART SOC Team - Week 2

## Project Overview
This repository contains Week 2 SOC (Security Operations Center) practical work including alert management, incident response, alert triage, evidence preservation, and a complete attack simulation (capstone project).

All tasks are documented with screenshots, analysis, and reports.

---

## 1. Alert Management Practice
- Created alert classification system using Google Sheets  
- Assigned priorities based on CVSS scoring  
- Mapped alerts with MITRE ATT&CK techniques  
- Visualized alerts using dashboard (pie chart)  
- Created incident ticket and escalation workflow  

---

## 2. Response Documentation
- Developed incident response template (SANS format)  
- Documented phishing incident with timeline and impact  
- Created investigation steps and checklist  
- Performed post-mortem analysis  
- Designed workflow using Draw.io  

---

## 3. Alert Triage Practice
- Simulated brute-force SSH attack detection  
- Analyzed logs showing multiple failed login attempts  
- Identified suspicious IP activity  
- Validated threat using VirusTotal and AlienVault OTX  
- Confirmed alert as a true positive  

---

## 4. Evidence Preservation
- Collected volatile data (network connections using netstat)  
- Captured memory dump from system  
- Generated SHA256 hash for integrity verification  
- Maintained chain of custody documentation  
- Performed basic forensic analysis  

---

## 5. Capstone Project (Full SOC Cycle)
- Simulated attack using Metasploit (VSFTPD exploit)  
- Detected attack using Wazuh  
- Performed alert triage and analysis  
- Responded by isolating system and blocking attacker IP  
- Generated report and stakeholder briefing  

---

## SOC Workflow
Attack → Detection → Triage → Response → Recovery → Reporting
